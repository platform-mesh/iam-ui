# Status Info Popover Component

A StatusInfoPopoverComponent displays status information and toggles between a list view and a detailed view of selected items.

## Features

Status Display: Uses fd-object-status to show status with icons, labels, and custom styles.
Dynamic Popover: Switches between a detailed item view and a list view.
Interactive Back Button: Appears in the detailed view if multiple items exist.

## How to use

```html
<dxp-status-info-popover [statusInfo]="statusInfo"> </dxp-status-info-popover>
```
