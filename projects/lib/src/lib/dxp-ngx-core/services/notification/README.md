# Notification Service

The Notification Service provides methods for displaying error and information messages.

The errors are shown on top as [Fundamentals message strip](https://fundamental-ngx.netlify.app/#/core/message-strip).

The information messages are shown on teh bottom as [Fundamentals message toast](https://fundamental-ngx.netlify.app/#/core/message-toast)

In both cases, the first letter of the displayed message will be uppercase.

## Dependencies

- `@luigi-project/client`
- `@luigi-project/client-support-angular`
- `@fundamental-ngx/core`
