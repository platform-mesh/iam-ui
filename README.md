# IAM-UI

## Description

The IAM-UI project provides the user interface for managing [DXP members](https://portal.hyperspace.tools.sap/projects/dxp/documentation/End-User).
It is based on [Angular](https://github.com/angular), uses [SAP Fundamental](https://github.com/SAP/fundamental-ngx) and [SAP Luigi](https://github.com/SAP/luigi).

## Installation

```bash
$ npm install
```

## Running the app

- This project uses `@dxp` libraries, to be able to run `npm install` follow the [Frontend Library Setup](https://portal.hyperspace.tools.sap/projects/dxp/team-docs/Development/Frontend-Library-Setup)
- See also [How to build, run and test](https://portal.hyperspace.tools.sap/projects/dxp/team-docs/Development/Frontend-General)

Then see the micro component within the sap dev by navigating to the [Hyperspace Portal](https://portal.d1.hyperspace.tools.sap/)

## Test

```bash
# unit tests
$ npm run test:unit

# test coverage
$ npm run test:cov
```
